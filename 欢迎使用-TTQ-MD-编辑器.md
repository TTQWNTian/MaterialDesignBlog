# 欢迎使用 TTQ MD 编辑器

这是一个基于 Material Design 3 的 Markdown 编辑器。

点击右上角"+"按钮创建新文档，或点击现有文档卡片进行编辑。

### 功能特色
- **多主题支持**：多种颜色主题和深色模式
- **本地存储**：所有文档保存在浏览器本地
- **全文搜索**：快速查找文档内容
- **分类管理**：按标签组织文档
- **GitHub 同步**：支持备份到云端指定文件夹（需配置）

## Markdown 基础语法

### 标题
```
# 一级标题
## 二级标题
### 三级标题
```

### 强调
```
**粗体文本**
*斜体文本*
~~删除线文本~~
```

### 列表
```
- 无序列表项
- 另一个列表项

1. 有序列表项
2. 第二个项目
```

### 代码
```javascript
// 代码块
function hello() {
    console.log("Hello, Markdown!");
}
```
